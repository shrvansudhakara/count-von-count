## ⚙️ Setup
1. Clone the repo 
```
git clone https://github.com/shrvansudhakara/count-von-count.git
```
2. Go to the project directory
```
cd count-von-count
```
3. Install dependencies
```
npm install
```

4. Start dev server
```
npm run dev
```

6. Live at `http://localhost:5173/`
